为什么晶体是几何的用代码模拟晶体生长结语

晶体的几何是宇宙给我们的一个暗示：原来完美是可以长出来的。一颗盐粒在显微镜下是立方体，一片雪花是六重对称，一束石英是六棱柱。这些形状不是被设计的，而是被"生长"出来的。

为什么晶体是几何的

答案藏在原子层面。当原子以最低能量状态排列时，它们会自然形成周期性结构，而周期性结构在宏观上就表现为规则的几何形状。

完美不是被画出来的，它是被允许发生的。

用代码模拟晶体生长

0
const grid = Array(100).fill(0).map(() => Array(100).fill(0));
grid[50][50] = 1; 1

for (let step = 0; step < 500; step++) {
  for (let x = 1; x < 99; x++) {
    for (let y = 1; y < 99; y++) {
      if (grid[x][y] === 0) {
        const n = grid[x-1][y] + grid[x+1][y] + grid[x][y-1] + grid[x][y+1];
        if (n > 0 && Math.random() < 0.3) grid[x][y] = 1;
      }
    }
  }
}

这段代码用最朴素的"邻居感染"逻辑生成晶体状的结构。改变 0.3 这个概率，你会得到从尖锐雪花到密实晶簇的不同形态。

结语

做创意编程的人，经常会迷失在"我要做出什么形状"的问题里。也许答案不是去想，而是去定义规则，然后让规则自己生长。晶体如是，好作品亦然。
